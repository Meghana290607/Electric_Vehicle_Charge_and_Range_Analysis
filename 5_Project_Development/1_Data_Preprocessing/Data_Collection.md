# Data Collection

## Introduction

Data collection is the first step in the development process. The project uses an Electric Vehicle (EV) dataset containing information about vehicle specifications, battery performance, charging infrastructure, and pricing.

## Dataset Contents

The dataset includes:

- Vehicle Brand
- Model
- Battery Capacity
- Driving Range
- Charging Time
- Charging Speed
- Top Speed
- Price
- Body Style
- Powertrain
- Charging Station Location
- Charging Station Type

## Data Source

The dataset was collected from publicly available EV data sources and organized into a structured format suitable for analysis.

## Purpose

The collected data serves as the foundation for creating interactive dashboards and generating meaningful insights into electric vehicle performance and charging infrastructure.
