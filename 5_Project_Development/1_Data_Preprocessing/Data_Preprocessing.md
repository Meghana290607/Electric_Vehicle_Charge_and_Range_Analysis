# Data Preprocessing

## Steps Performed

1. Imported the dataset.
2. Inspected the data for inconsistencies.
3. Removed duplicate rows.
4. Filled or removed missing values.
5. Corrected invalid records.
6. Standardized attribute names.
7. Formatted numerical and text values.
8. Verified the final dataset.

## Advantages

- Improved data quality.
- Reduced redundancy.
- Enhanced visualization accuracy.
- Faster query execution.
