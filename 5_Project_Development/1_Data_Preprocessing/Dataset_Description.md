# Dataset Description

The Electric Vehicle dataset contains information required for analyzing EV performance and charging infrastructure.

## Key Attributes

- Brand
- Model
- Battery Capacity
- Driving Range
- Charging Time
- Charging Speed
- Top Speed
- Price
- Body Style
- Powertrain
- Charging Station Region
- Charging Station Type

## Purpose

These attributes are used to create interactive Tableau dashboards for comparing electric vehicles and analyzing charging trends.
