# Data Cleaning

## Objective

The objective of data cleaning is to improve the quality of the dataset before importing it into the database.

## Data Cleaning Activities

- Removed duplicate records.
- Handled missing values.
- Standardized column names.
- Corrected inconsistent values.
- Converted numerical columns to appropriate data types.
- Removed unwanted spaces and null entries.

## Benefits

- Improved data accuracy.
- Better dashboard performance.
- Reliable visualizations.
- Consistent analysis results.

## Outcome

The cleaned dataset was successfully prepared for storage in the MySQL database.
