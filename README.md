# 🚗 Visualization Tool for Electric Vehicle Charge and Range Analysis

## Project Overview

The **Visualization Tool for Electric Vehicle Charge and Range Analysis** is a Business Intelligence and Data Analytics project developed to analyze and visualize Electric Vehicle (EV) data using interactive dashboards and a web application. The project integrates **MySQL**, **Tableau Public**, and **Python Flask** to provide meaningful insights into EV performance, charging infrastructure, battery capacity, driving range, pricing, and other vehicle specifications.

The application enables users to explore EV data through interactive dashboards, maps, charts, and stories, supporting informed decision-making for consumers, researchers, and organizations.

---

## Problem Statement

With the rapid growth of Electric Vehicles, large volumes of data are generated related to vehicle specifications, charging infrastructure, battery performance, and pricing. Traditional reports and spreadsheets make it difficult to analyze and compare this information effectively.

This project addresses the challenge by providing an interactive visualization platform that allows users to explore and analyze Electric Vehicle data through dynamic dashboards and stories.

---

## Objectives

- Analyze Electric Vehicle data using interactive visualizations.
- Compare vehicle specifications such as battery capacity, driving range, and top speed.
- Visualize charging station distribution across different regions.
- Analyze pricing trends of electric vehicles.
- Develop an interactive Tableau Dashboard and Story.
- Build a Flask-based web application to display visualizations.
- Support data-driven decision-making using Business Intelligence tools.
