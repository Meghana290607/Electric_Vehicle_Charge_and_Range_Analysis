# Project Milestones

## Milestone 1
### Problem Understanding and Requirement Analysis
- Identify the project objectives.
- Define the problem statement.
- Gather business and technical requirements.
- Conduct literature survey.
- Identify stakeholders and project scope.

---

## Milestone 2
### Data Collection and Analysis
- Collect Electric Vehicle dataset.
- Analyze dataset attributes.
- Identify useful features.
- Understand charging infrastructure and vehicle specifications.

---

## Milestone 3
### Data Preprocessing and Database Development
- Remove duplicate records.
- Handle missing values.
- Standardize data formats.
- Import cleaned dataset into MySQL.
- Execute SQL queries for analysis.

---

## Milestone 4
### Tableau Dashboard and Story Development
- Create Tableau worksheets.
- Design interactive dashboard.
- Develop Tableau Story.
- Apply filters and calculated fields.
- Generate visual insights.

---

## Milestone 5
### Flask Web Application Development
- Develop Flask application.
- Design Home, About, Dashboard, Story, Team, and Contact pages.
- Embed Tableau Dashboard.
- Embed Tableau Story.
- Improve user interface and navigation.

---

## Milestone 6
### Testing and Validation
- Perform functional testing.
- Verify dashboard accuracy.
- Test responsiveness.
- Fix identified bugs.
- Validate application performance.

---

## Milestone 7
### Documentation and Demonstration
- Prepare project documentation.
- Create GitHub repository.
- Record project demonstration video.
- Prepare presentation slides.
- Complete project report.

---

## Milestone 8
### Project Completion and Deployment
- Deploy the Flask application.
- Publish Tableau Dashboard and Story.
- Upload project to GitHub.
- Perform final review.
- Submit the completed project.
