## Dashboard Description

The dashboard visualizes:

- EV Charging Stations in India
- Charging Stations by Region
- Charging Stations by Type
- Top Speed Analysis
- Price Analysis
- Battery Capacity
- Driving Range
- Top 10 EV Brands
- Powertrain Analysis
- Body Style Distribution

Interactive filters allow users to explore the dataset efficiently.
