# Tableau Workbook

## Overview

This folder contains the Tableau workbook and supporting resources used in the Visualization Tool for Electric Vehicle Charge and Range Analysis project.

## Contents

- Tableau Workbook (.twbx)
- Dashboard Screenshots
- Tableau Dashboard Link
- Tableau Story Link

## Dashboard Features

- Interactive Dashboard
- Tableau Story
- KPI Cards
- Maps
- Charts
- Filters
- Dynamic Visualizations

The Tableau dashboard enables users to analyze electric vehicle specifications and charging infrastructure through interactive visualizations.
