# Workflow

## Introduction

The workflow describes the sequence of activities involved in developing the Visualization Tool for Electric Vehicle Charge and Range Analysis.

---

## Step 1: Data Collection

Collect the electric vehicle dataset from publicly available sources.

---

## Step 2: Data Cleaning

Perform preprocessing by:

- Removing duplicate values
- Handling missing values
- Standardizing formats
- Validating records

---

## Step 3: Store Data

Import the cleaned dataset into MySQL.

---

## Step 4: Data Analysis

Write SQL queries to retrieve the required information.

---

## Step 5: Tableau Visualization

Create worksheets such as:

- Charging Station Map
- Price Analysis
- Range Analysis
- Battery Capacity Analysis
- Brand Comparison
- Powertrain Distribution

---

## Step 6: Dashboard Development

Combine worksheets into a single interactive dashboard.

---

## Step 7: Story Development

Create a Tableau Story explaining the insights.

---

## Step 8: Flask Integration

Embed the dashboard and story into the Flask application.

---

## Step 9: Testing

Test dashboard loading, navigation, responsiveness, and functionality.

---

## Step 10: Deployment

Publish the project using GitHub and Tableau Public.
