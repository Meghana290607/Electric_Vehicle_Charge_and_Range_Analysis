# Design Decisions

## Why MySQL?

- Structured data storage
- Fast query execution
- Reliable database management

---

## Why Tableau?

- Interactive dashboards
- Powerful visualizations
- Story creation
- Easy filtering

---

## Why Flask?

- Lightweight framework
- Easy integration with Tableau
- Fast deployment
- Simple routing

---

## Why HTML/CSS/JavaScript?

- Responsive web interface
- Attractive UI
- Better user experience

---

## Why Bootstrap?

- Mobile responsiveness
- Faster frontend development
- Consistent design

---

## Design Principles

- Simplicity
- Scalability
- Reusability
- Maintainability
- Performance
- User-Friendliness

---

## Conclusion

The selected technologies complement each other and provide a complete end-to-end solution for data storage, analysis, visualization, and deployment.
