# SQL

## Overview

This folder contains all SQL scripts used in the Visualization Tool for Electric Vehicle Charge and Range Analysis project.

## Files

- Database_Creation.sql
- Create_Tables.sql
- Analysis_Queries.sql

## Purpose

The SQL scripts were used to:

- Create the project database
- Create database tables
- Store EV data
- Retrieve analytical information
- Prepare data for Tableau dashboards

The database management system used in this project is **MySQL**.
